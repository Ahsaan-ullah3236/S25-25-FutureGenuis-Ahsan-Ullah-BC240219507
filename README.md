# AccessForm – Web Accessible Form Builder
S25-FutureGenius-AhsanUllah_bc240219507

##  Project Overview

AccessForm is a web-based form builder application developed as a Final Year Project.  
This system focuses on accessibility and follows WCAG 2.1 guidelines to make forms usable for all users, including Persons with Disabilities (PWD).

The system allows users to create forms, share them, and collect responses in an accessible way.

---

##  Main Purpose

Many form builder applications do not fully support accessibility features like:

- Keyboard navigation
- Screen reader compatibility
- High contrast mode

AccessForm solves this problem by providing an accessible and user-friendly form system.

##  User Roles

- Admin – Manage users and forms
- Form Creator – Create and manage forms
- Respondent – Fill and submit forms

---

##  Main Features

- User Login & Signup
- Dashboard for each role
- Create forms with:
  - Text field
  - Email field
  - Number field
  - Checkbox
  - Radio button
  - Dropdown
- Form Preview
- Store responses in database
- Responsive design
- Basic accessibility support

---

##  Accessibility Features

- Keyboard navigation support
- Proper labels for form fields
- ARIA attributes
- Responsive layout

##  How to Run the Project

1. Install XAMPP
2. Copy project folder into:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Import database file
5. Open in browser:
   http://localhost/project-folder-name
